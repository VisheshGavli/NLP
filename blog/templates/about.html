Web Blogging

Being a Python Enthusiast, have built this "Web Blogging" Project to demostrate my Web Development expertise using Flask (Python-Based Web Framework). This project aims to explain the usage of open-source technologies and how these technologies interact between each other. Further I have listed all the technologies I have used and mentioned a short description for each.

Technology Stack

1. Backend

Flask
Flask is a high-level Python Web framework. I have chosen it, because it is Python based, you can write fast, you don’t have to repeat yourself and it has a really good documentation.

2. Graphical User Interface

Twitter Bootstrap
Bootstrap is sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development. It is purely written in HTML5/CSS3. This framework is the most popular project at Github at the moment. Nowadays a lot of web pages looks very similar, because of Bootstrap popularity.

JQuery
JQuery is a fast, small, and feature-rich JavaScript library. This is a dependency for Twitter Bootstrap.

3. Authentication

Flask-Login
Flask-Login provides user session management for Flask. It handles the common tasks of logging in, logging out, and remembering your users’ sessions over extended periods of time. The most important part of an application that uses Flask-Login is the LoginManager which handles task like loading of User and where to send users once they log in.

4. Database

SQLite
SQLite is a software library that implements a self-contained, serverless, zero-configuration, transactional SQL database engine. It is very easy to use for development purpose.It works very good, when you have to develop in a team. Additionally the SQLAlchemy ORM (Object Relational Mapper) library facilitates communication with database engines and provides python-based SQL expressions for database manupilation tasks.

5. Content Rendering

Jinja2
Flask uses the Jinja2 templating engine to render the HTML content via the HTTPs response.
